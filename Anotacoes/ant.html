Exemplo de atalho para inserir várias tags (podendo ser uma dentro da outra):
Inserindo várias tags DIVs:
div*8 = Vai inserir 8 tags divs em sequencia.
.numero$*4>.letras#$*3


Separação das tags DIVs de acordo com as boas práticas:
DIV header = Para o cabeçalho do site
DIV main = Para o conteudo principal do site
DIV Footer = Para o Rodapé do site.
